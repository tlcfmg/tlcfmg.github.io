surveychat is an open-source web application for researchers who want to administer surveys or randomized experiments with LLM-based conversational agents. Configure one chatbot for all participants or multiple chatbot conditions for experimental designs, then export anonymized JSON transcripts back to a parent survey platform such as Qualtrics.

Learn more in the [surveychat documentation](https://surveychat.github.io/) or view the [source code on GitHub](https://github.com/surveychat/surveychat).
